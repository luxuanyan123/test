理解 for(let i = 0; i < 10; i++) 的工作原理

For循环解析

for(let i = 0; i < 10; i++) { console.log("你好"); }
1

初始化 (let i = 0)

在循环开始前执行一次，创建一个计数器变量 i 并初始化为 0。

2

条件检查 (i < 10)

每次循环迭代前检查条件。如果为真，则执行循环体；如果为假，则终止循环。

3

循环体执行

当条件为真时，执行大括号 {} 内的所有代码。

4

更新表达式 (i++)

每次循环体执行后执行，更新计数器变量（这里是将 i 增加 1）。

关键点

  • 循环会执行10次（i从0到9）
  • 当i等于10时，条件i < 10为假，循环终止
  • i++ 等同于 i = i + 1
  • 循环结束后，变量i的值是10
