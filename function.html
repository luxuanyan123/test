JavaScript 函数执行流程分析

理解 return 语句如何影响代码执行

代码分析

function add(a, b) { let sum = a + b; return sum; // 函数在此处结束并返回值 console.log(sum); // 这行代码永远不会执行 } console.log(add(2, 3)); // 输出: 5
1

函数定义

创建名为 add 的函数，它接受两个参数 a 和 b。

2

函数调用

执行 add(2, 3)，将值 2 和 3 传递给函数。

3

变量计算

在函数内部，计算 sum = a + b → 2 + 3 = 5。

4

return 语句执行

执行 return sum，函数结束并返回值 5。

重要：return 之后的代码不会执行！

5

控制台输出

console.log(add(2, 3)) 接收返回值 5 并输出到控制台。

关键点

  • return 语句会立即终止函数执行
  • return 之后的代码永远不会执行
  • 函数调用表达式 add(2,3) 会被替换为返回值（5）
  • 因此 console.log(sum) 永远不会执行
  • 最终控制台只输出一个值：5
